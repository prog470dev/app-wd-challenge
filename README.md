## 盛り込んだ指定オプショナル項目

* 募集一覧をタップした後の詳細画面

## 追加機能
* リスト状態で関連するキーワードをロングタップで表示
    - 詳細画面を開くよりも手軽な操作で確認可能
    - 詳細説明の文章から予め設定されたキーワード（現在は言語のみ）を抽出
